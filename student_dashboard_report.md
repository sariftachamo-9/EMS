# 📘 Student Dashboard Report (V2)

This report detail the specific features and interface elements available to **Active Students** in the EMS portal.

---

## 🏗️ 1. Navigation & Access
*   **Breadcrumb**: `Staff Dashboard > Student`
*   **Primary Menu**: `Dashboard`, `My Profile`, `My Queries`, `Leaves`, `My Fees`.

---

## ⚡ 2. The Command Center (Attendance Hub)
*   **Daily Timer**: Keeps track of daily workshop or lecture attendance.
*   **Session Management**:
    *   **Emerald**: Logged in to session.
    *   **Rose**: Leaving daily session.
*   **Break Management**: Standard break and resuming logic.
*   **Heartbeat**: Ensures real-time presence verification.

---

## 📈 3. Student-Specific Insight Grid
*   **Attendance Score**: Continuous percentage based on scheduled sessions.
*   **Leave Balance**: Number of permissible absences allowed.
*   **Workshop Milestone**: Shows "Ongoing" progress in the current course.
*   **💰 Fee Status Tracking**:
    *   Direct link to 'My Fees' for payment verification.
    *   Shows active enrollment status.

---

## 🗓️ 4. Activity & Communications
*   **Daily Attendance Log**: FullCalendar tracking.
*   **Absent Saturday Highlight**: Soft red indicator for missed Saturday sessions.
*   **Notice Board**: Important academic or administrative updates.

---

## 🛡️ 5. Security Hub
*   **Boot ID Integrity**: Session reset after each server restart.
*   **Digital Access Badge**: QR Code for campus and building access.
