# 🎓 Intern Dashboard Report (V2)

This report detail the specific features and interface elements available to **Active Interns** in the EMS portal.

---

## 🏗️ 1. Navigation & Access
*   **Breadcrumb**: `Staff Dashboard > Intern`
*   **Primary Menu**: `Dashboard`, `My Profile`, `My Queries`, `Leaves`, `Stipends`.

---

## ⚡ 2. The Command Center (Attendance Hub)
*   **Attendance Clock**: Precise counter for daily intern hours.
*   **Check-in/Out Logic**:
    *   **Emerald**: Intern present.
    *   **Rose**: Intern leaving.
*   **Break Management**: Full support for break pauses.
*   **Heartbeat**: Active session verification.

---

## 📈 3. Intern-Specific Insight Grid
*   **Attendance Score**: Tracked as active days vs programmed ones.
*   **Leave Balance**: Count of allowed time-off during the internship.
*   **Program Milestone**: Displays "Active" during the workshop/internship period.
*   **💰 Stipend Tracking**:
    *   Access to stipend history (via the Stipends tab).
    *   Shows active status in the program.

---

## 🗓️ 4. Activity & Communications
*   **FullCalendar Log**: Detailed record of daily activities and attendance.
*   **Absent Saturday Highlight**: Identifies any Saturdays missed if they were part of the program schedule.
*   **Notice Board**: Access to departmental and company-wide notices.

---

## 🛡️ 5. Security Hub
*   **Boot ID Reset**: Forcing re-login after every server update for security.
*   **QR Security Badge**: Digital ID for office or workshop entrance.
